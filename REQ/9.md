Extend the scraper (#4) to include a curated set of **12 high‑value sources** for detecting human rights violations, IHL breaches, and political violence in **Colombia**, with a special focus on the **Putumayo region**. These sources range from local media and grassroots organizations to national outlets and international bodies.

The goal is to capture structured data (articles, communiqués, reports) that can be used to generate pre‑alerts and enrich the evidence base for documented cases.

**Simplification for MVP:**
- Priority is given to sources with RSS feeds (easy integration).
- Web scraping sources are limited to those with consistent HTML structure.
- Facebook is accessed via the official Graph API (no HTML scraping).
- PDF extraction is kept minimal (just for the Defensoría and CIDH when strictly needed).

**Post‑MVP:** Add more sources (e.g., Twitter/X accounts, other regional media) and improve filtering and deduplication.

## Integration Strategy

### 1. RSS Feeds (Priority 1)
- **Sources:** INDEPAZ (#3) ✅, MiPutumayo Noticias (#6) ✅, ReliefWeb oPt (#7) ✅
- **Implementation:** Use `rss-parser` to fetch and parse feeds. No special authentication required.

### 2. Facebook Graph API (Priority 2)
- **Source:** Red DDHH Putumayo (#2)
- **Implementation:**
  - Obtain `page_id` from the Graph API.
  - Use `GET /{page-id}/posts?fields=message,created_time,permalink_url` with a valid access token.
  - Store the post text as `clean_text` and the post URL as `url`.
- **Note:** Respect Facebook’s rate limits and terms of service.

### 3. Web Scraping (Remaining Sources)
- **Sources:** Conexión Putumayo (#1), El Espectador (#4), Comisión Intereclesial (#5), HRW (#8), CIDH (#9), ONU‑DH (#10), Defensoría (#11), Resumen Latinoamericano (#12), MiPutumayo Noticias (#6)
- **Implementation:**
  - Use `axios` + `cheerio` for HTML parsing.
  - For each source, define:
    - `listSelector`: CSS selector for article links on the main page.
    - `titleSelector`: CSS selector for article title.
    - `dateSelector`: CSS selector for publication date.
    - `bodySelector`: CSS selector for article body text.
    - `cleanText`: Use `cleanHTML()` (from #4) to strip tags and extract plain text.
  - **Geographic Filter:** For national sources (#4, #8, #9, #10, #11, #12), apply a **geographic filter** for `Putumayo`, `Mocoa`, `Puerto Asís`, and other municipalities. For other regions, you can optionally expand filters in the future.

### 4. PDF Extraction (Defensoría del Pueblo)
- **Source:** Defensoría del Pueblo (#11)
- **Implementation:** If an Early Warning or report is in PDF format, use `pdf-parse` to extract text. Store the parsed text as `clean_text` and the original PDF URL as `url`.

### 3. Facebook Graph API Client

Create `lib/facebook.ts`:

```typescript
import axios from 'axios';

const FB_ACCESS_TOKEN = process.env.FB_ACCESS_TOKEN;
const FB_PAGE_ID = process.env.FB_PAGE_ID;

export async function fetchFacebookPosts(limit = 10) {
  const url = `https://graph.facebook.com/v19.0/${FB_PAGE_ID}/posts`;
  const params = {
    access_token: FB_ACCESS_TOKEN,
    fields: 'message,created_time,permalink_url',
    limit: limit,
  };
  const response = await axios.get(url, { params });
  return response.data.data;
}
```

function isRelevantForPutumayo(text: string): boolean {
