## Description

Sync the `sivel3agent` with the Banco de Datos del CINEP / Noche y Niebla database (60,000+ cases). This serves three purposes:

1. **Deduplication**: Avoid generating pre‑alerts for cases already documented
2. **Enrichment**: Add new information to existing cases when new sources appear
3. **Model feedback**: Use real cases to improve the LLM extraction (#11)

**Sync frequency:**
- Initial: one‑time full import (60,000+ cases)
- Incremental: every 6 months (when Noche y Niebla publishes new cases)

---
