## Description

Implement a **resumable historical crawler** that retrieves articles from **July 2025 to present** from key DDHH sources by iterating over their paginated archives. This complements the RSS scraper (#4/#9/#12) which only returns the most recent 10‑20 items.

**Why this is needed:**
- RSS feeds only return the latest articles. To have a meaningful dataset for pre‑alert generation, we need coverage back to 2025‑07‑01 (the cutoff date for the MVP).
- Some sources (Comisión Intereclesial, Verdad Abierta, INDEPAZ) have thousands of relevant articles from this period archived on their sites.
- Web scraping is required because these sources do not expose historical RSS archives.
- Historical archives can be large and take hours/days to fully crawl, so we need to be able to pause and resume without losing progress.
- We must avoid re‑processing the same content when the crawler runs again.

**Scope for MVP:** Focus on the 5 highest‑density DDHH sources that have structured archives. Additional sources can be added to the queue by the operator.

## Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Queue‑based architecture** | Each site is a job in the queue. Jobs can be added manually or automatically. |
| **Resumable crawling** | Each site job tracks its progress (`last_offset`, `last_month`, `last_page`). |
| **Time‑bound execution** | Each run has a time limit (e.g., 10 minutes). The crawler stops when time is up and resumes next run. |
| **Incremental updates** | On subsequent runs, only new content is fetched (check `last_fetched_at`). |
| **Deduplication** | Content is stored with `url` and `content_hash` unique constraints. |

## Data Model

### 1. `crawler_queue` — Job queue for historical sources

```sql
CREATE TABLE crawler_queue (
    id SERIAL PRIMARY KEY,
    source_name VARCHAR(100) NOT NULL,
    source_type VARCHAR(20) NOT NULL,      -- 'wordpress', 'spip', 'sitemap', 'custom'
    base_url TEXT NOT NULL,
    config JSONB,                           -- { listSelector, bodySelector, dateSelector, etc. }
    status VARCHAR(20) DEFAULT 'pending',   -- pending, running, paused, completed, failed
    priority INTEGER DEFAULT 0,
    progress JSONB,                         -- { last_month: '2025-07', last_page: 2, last_offset: 0, total_processed: 0 }
    last_run_at TIMESTAMP,
    next_run_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_crawler_queue_status ON crawler_queue(status);
CREATE INDEX idx_crawler_queue_priority ON crawler_queue(priority DESC);
```

## Crawling Strategy

### WordPress Sites (INDEPAZ, Verdad Abierta, Comisión Intereclesial, MiPutumayo)

WordPress standard archive URLs: `https://example.com/YYYY/MM/`

For each month from 2025‑07 to current:
1. Fetch `https://example.com/YYYY/MM/`
2. Parse HTML to extract article links (`<article> h2 a`, or similar)
3. Handle pagination: `?paged=2`, `?paged=3`, etc.
4. For each article link, fetch the full page
5. Extract title (`<h1>`), date (`<time>`), and full text from article body
6. Clean HTML → `clean_text`
7. Store in `source` table (matches existing scraper schema)

**With resume support:** Track `last_month` and `last_page` in `progress`.

### SPIP Sites (Prensa Rural)

SPIP uses offset‑based pagination: `spip.php?page=backend&debut_articles=N`

1. Start with `debut_articles=0`
2. Fetch page, parse list of articles
3. For each article link, fetch full page
4. Extract content from SPIP selectors
5. Continue with `debut_articles+=10` until articles are before 2025‑07

**With resume support:** Track `last_offset` in `progress`.

## Implementation

### 1. Generic WordPress Archive Crawler (with resume support)

Create `lib/crawlWordPress.ts`:

```typescript
// lib/crawlWordPress.ts
import axios from 'axios'
import { parse as parseHTML } from 'node-html-parser'

interface ArchiveArticle {
  url: string
  title: string
  date: Date | null
  fullText: string
  contentHash: string
}

export async function crawlWordPressArchive(
  baseUrl: string,
  progress: { last_month?: string; last_page?: number } = {},
  listSelector = 'article h2 a',
  titleSelector = 'h1',
  dateSelector = 'time',
  bodySelector = 'div.entry-content, article .content, .post-content',
): Promise<ArchiveArticle[]> {
  const articles: ArchiveArticle[] = []
  const startMonth = progress.last_month || '2025-07'
  const startPage = progress.last_page || 1

  const months = getMonthsFrom(startMonth)

  for (const month of months) {
    let page = (month === startMonth) ? startPage : 1
    while (true) {
      const archiveUrl = `${baseUrl}/${month}/page/${page}/`
      console.log(`  Crawling ${archiveUrl}…`)

      try {
        const { data } = await axios.get(archiveUrl, { timeout: 15000 })
        const root = parseHTML(data)

        // Extract article links from archive page
        const links = root.querySelectorAll(listSelector)
        if (links.length === 0) break

        for (const link of links) {
          const href = link.getAttribute('href')
          if (!href) continue

          try {
            const fullArticle = await fetchArticle(href, titleSelector, dateSelector, bodySelector)
            if (fullArticle) articles.push(fullArticle)
          } catch {
            console.warn(`    Failed to fetch: ${href}`)
          }
        }

        // Check for next page
        const nextPage = root.querySelector('.next, .nav-links .next')
        if (!nextPage) break
        page++
      } catch {
        break
      }
    }
  }

  return articles
}

function getMonthsFrom(startMonth: string): string[] {
  const months: string[] = []
  const [startYear, startMon] = startMonth.split('-').map(Number)
  let year = startYear
  let month = startMon
  const now = new Date()
  const currentYear = now.getFullYear()
  const currentMonth = now.getMonth() + 1

  while (year < currentYear || (year === currentYear && month <= currentMonth)) {
    months.push(`${year}/${String(month).padStart(2, '0')}`)
    month++
    if (month > 12) { month = 1; year++ }
  }
  return months
}

async function fetchArticle(
  url: string,
  titleSelector: string,
  dateSelector: string,
  bodySelector: string,
): Promise<ArchiveArticle | null> {
  const { data } = await axios.get(url, { timeout: 10000 })
  const root = parseHTML(data)

  const titleEl = root.querySelector(titleSelector)
  const dateEl = root.querySelector(dateSelector)
  const bodyEl = root.querySelector(bodySelector)

  if (!titleEl || !bodyEl) return null

  const title = titleEl.textContent?.trim() || ''
  const date = dateEl?.getAttribute('datetime')
    ? new Date(dateEl.getAttribute('datetime')!)
    : null
  const fullText = cleanHTML(bodyEl.innerHTML || bodyEl.textContent || '')
  const contentHash = createHash('sha256').update(fullText).digest('hex')

  return { url, title, date, fullText, contentHash }
}

function cleanHTML(html: string): string {
  const root = parseHTML(html)
  root.querySelectorAll('script, style, nav, footer, .comments').forEach(el => el.remove())
  return (root.textContent || '').replace(/\s+/g, ' ').trim()
}
```

### 2. SPIP Archive Crawler (with resume support)

Create `lib/crawlSPIP.ts`:

```typescript
// lib/crawlSPIP.ts
import axios from 'axios'
import { parse as parseHTML } from 'node-html-parser'
import { createHash } from 'crypto'

export async function crawlSPIPArchive(
  baseUrl: string,
  progress: { last_offset?: number } = {},
): Promise<ArchiveArticle[]> {
  const articles: ArchiveArticle[] = []
  const CUTOFF = new Date('2025-07-01')
  let offset = progress.last_offset || 0
  const BATCH_SIZE = 10

  while (true) {
    const url = `${baseUrl}?debut_articles=${offset}`
    console.log(`  Crawling SPIP offset ${offset}…`)

    try {
      const { data } = await axios.get(url, { timeout: 15000 })
      const root = parseHTML(data)

      const links = root.querySelectorAll('a[href*="spip.php?article"]')
      if (links.length === 0) break

      for (const link of links) {
        const href = link.getAttribute('href')
        if (!href) continue

        try {
          const articleData = await axios.get(href, { timeout: 10000 })
          const articleRoot = parseHTML(articleData.data)

          const title = articleRoot.querySelector('h1')?.textContent?.trim() || ''
          const dateStr = articleRoot.querySelector('.date, time')?.textContent?.trim()
          const date = dateStr ? new Date(dateStr) : null
          const body = articleRoot.querySelector('.texte, .entry-content, article')
          const fullText = body?.textContent?.replace(/\s+/g, ' ').trim() || ''
          const contentHash = createHash('sha256').update(fullText).digest('hex')

          if (date && date >= CUTOFF) {
            articles.push({ url: href, title, date, fullText, contentHash })
          }
        } catch {
          console.warn(`    Failed to fetch article: ${href}`)
        }
      }

      // Check if we're done
      offset += BATCH_SIZE
    } catch {
      break
    }
  }

  return articles
}
```

### 3. Historical Crawler Script (with time limit and resume)

Create `scripts/crawl-history.ts`:

```typescript
// scripts/crawl-history.ts
import { newKyselyPostgresql } from '../.config/kysely.config.js'
import { crawlWordPressArchive } from '../lib/crawlWordPress'
import { crawlSPIPArchive } from '../lib/crawlSPIP'
import { classifyByKeywords } from '../lib/classifyByKeywords'

const MAX_RUN_TIME = 10 * 60 * 1000 // 10 minutes

async function crawl() {
  const db = newKyselyPostgresql()
  const startTime = Date.now()

  // Get next pending job (highest priority, oldest first)
  const job = await db
    .selectFrom('crawler_queue')
    .selectAll()
    .where('status', '=', 'pending')
    .orderBy('priority', 'desc')
    .orderBy('created_at', 'asc')
    .limit(1)
    .executeTakeFirst()

  if (!job) {
    console.log('No pending jobs in queue.')
    return
  }

  console.log(`Starting job: ${job.source_name} (${job.base_url})`)

  // Mark as running
  await db
    .updateTable('crawler_queue')
    .set({ status: 'running', last_run_at: new Date() })
    .where('id', '=', job.id)
    .execute()

  let progress = job.progress || {}
  let totalProcessed = progress.total_processed || 0

  try {
    // Execute the appropriate crawler based on source_type
    let articles = []
    if (job.source_type === 'wordpress') {
      articles = await crawlWordPressArchive(job.base_url, progress)
    } else if (job.source_type === 'spip') {
      articles = await crawlSPIPArchive(job.base_url, progress)
    } else if (job.source_type === 'sitemap') {
      articles = await crawlSitemap(job.base_url, progress)
    }

    // Save articles to source table
    for (const article of articles) {
      // Dedup by URL
      const exists = await db
        .selectFrom('source')
        .select('id')
        .where('url', '=', article.url)
        .executeTakeFirst()
      if (exists) continue

      const kw = classifyByKeywords(article.fullText, article.title)
      const isRelevant = kw.confidence === 'high' ? kw.relevant : null

      await db
        .insertInto('source')
        .values({
          url: article.url,
          medium: job.source_name,
          title: article.title,
          published_at: article.date,
          content_hash: article.contentHash,
          clean_text: article.fullText,
          is_relevant: isRelevant,
          classification_reason: kw.confidence === 'high' ? `[kw] ${kw.reason}` : null,
        })
        .execute()

      totalProcessed++
      console.log(`  📥 ${article.title.slice(0, 80)}`)
    }

    // Update progress
    progress = { ...progress, total_processed: totalProcessed }

    // Check if time limit exceeded
    const elapsed = Date.now() - startTime
    if (elapsed >= MAX_RUN_TIME) {
      console.log(`Time limit reached (${elapsed}ms). Pausing job.`)
      await db
        .updateTable('crawler_queue')
        .set({
          status: 'pending',
          progress: progress,
        })
        .where('id', '=', job.id)
        .execute()
      return
    }

    // If we've processed all articles, mark as completed
    const isComplete = await checkIfComplete(job, progress)
    if (isComplete) {
      await db
        .updateTable('crawler_queue')
        .set({
          status: 'completed',
          progress: progress,
        })
        .where('id', '=', job.id)
        .execute()
      console.log(`Job completed: ${job.source_name}`)
    } else {
      await db
        .updateTable('crawler_queue')
        .set({
          status: 'pending',
          progress: progress,
        })
        .where('id', '=', job.id)
        .execute()
    }

  } catch (error) {
    console.error(`Error in job ${job.source_name}:`, error)
    await db
      .updateTable('crawler_queue')
      .set({
        status: 'failed',
        progress: progress,
      })
      .where('id', '=', job.id)
      .execute()
  }
}

crawl()
  .then(() => process.exit(0))
  .catch((e) => { console.error(e); process.exit(1) })
```

// scripts/crawler-queue.ts
import { db } from '../lib/db'

async function addJob() {
  // CLI args: --add --name "..." --type "..." --url "..." [--config "..."]
  const { name, type, url, config } = parseArgs()

  await db
    .insertInto('crawler_queue')
    .values({
      source_name: name,
      source_type: type,
      base_url: url,
      config: config ? JSON.parse(config) : null,
      priority: 10, // user‑added sources get high priority
      status: 'pending',
    })
    .execute()

  console.log(`Added job: ${name}`)
}

async function listJobs() {
  const jobs = await db
    .selectFrom('crawler_queue')
    .selectAll()
    .orderBy('priority', 'desc')
    .execute()

  console.table(jobs)
}

async function resetJob(id: number) {
  await db
    .updateTable('crawler_queue')
    .set({ status: 'pending', progress: {} })
    .where('id', '=', id)
    .execute()

  console.log(`Reset job ${id}`)
}
